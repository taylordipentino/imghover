IMG Hover

A tiny jQuery plugin for adding hover effects to images.

Basic Usage

Applying the default options to a collection: 

$('img').imgHover();

Zoom Effect Usage

Disable the other default effects and change the default zoom scale: 

$('img').imgHover({radius: '0.5rem', grow: false, zoomScale: 1.2});

Grow Effect Usage

Disable the other default effects and change the default grow scale: 

$('img').imgHover({radius: '0.5rem', zoom: false, growScale: 1.07});

You can also shrink the image by passing a negative grow scale value: 

$('img').imgHover({radius: '0.5rem', zoom: false, growScale: 0.98});

Change Border Radius

If you have a border radius on your images, you need to pass the value as a parameter so it is applied to the wrapper div as well. This is set to false (disabled) by default, but you can change it to any valid property value (as a string) of the CSS border-radius property: 

$('img').imgHover(radius: '50%'});

Explicit Sizing

The default styling applies 100% width and automatic height to the image container and 100% width and height to the image. If you wish to change this you may pass an object with properties for both width and height to override the styles applied to the container: 

$('img').imgHover(size: {width: '50%', height: 'auto'}});

Touch Events

By default, touch events are used so the effects will work on touch devices. If you wish to disable this, set the touch option to false: 

$('img').imgHover({touch: false});
